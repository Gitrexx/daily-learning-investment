# Topic 29：最大回撤（Max Drawdown）与「扛得住」的心理学——账面腰斩时，你真实会做出什么动作

> **本篇目标**：上一篇（Topic 28）我们确认了一件残酷的事——**系统性风险躲不掉**，买遍全世界也压不穿那条「地板」。那么当地板真的塌下来、账户从高点被腰斩时，问题就从**数学**变成了**人性**。
>
> 这一篇要装进你脑子里的，是两把尺子和一个鸿沟。第一把尺子叫 **最大回撤（Maximum Drawdown, MDD）**：它衡量「从最高点跌到最低点，你的账户最深被砸出多大一个坑」。第二把尺子叫 **回本非对称性**：跌下去和爬回来，**根本不是一回事**——跌 50% 只需要「腰斩」，爬回来却要 **翻倍（+100%）**。而那个鸿沟，是**你以为你能扛的**和**你真到坑底会做出的动作**之间的落差——绝大多数人亏钱，不是死于不懂，而是死于**在坑底割了肉**。

---

## 一、先给「坑」下一个精确定义：什么是最大回撤

波动率（Topic 27）衡量的是「平时颠得有多厉害」，但它对涨和跌**一视同仁**。可你心里清楚：**账户涨 20% 的爽，和跌 20% 的痛，完全不成比例。** 我们真正怕的，不是「上下颠」，而是「**从高点一路往下、深不见底的那个坑**」。

**最大回撤**就是专门量这个坑的尺子：

> **最大回撤 = 从某个历史最高点（峰顶），跌到之后某个最低点（谷底）的最大跌幅百分比。**

$$ \text{MDD} = \frac{\text{谷底价值} - \text{峰顶价值}}{\text{峰顶价值}} = \min_t \frac{V_t - \max_{s\le t} V_s}{\max_{s\le t} V_s} $$

它永远是个**负数**（或 0）。举个例子：你的组合从 $10{,}000$ 涨到峰值 $12{,}000$，随后一路跌到谷底 $7{,}200$，再慢慢回升——那么这轮最大回撤 = $(7200-12000)/12000 = -40\%$。

注意三个关键点：

- **回撤是从「峰顶」量起，不是从你的成本量起。** 哪怕你整体还赚着钱，只要账户从阶段高点缩水，就算回撤。它量的是「**你曾经拥有过、又眼睁睁看着蒸发掉**」的那部分——而这恰恰是**心理上最难受**的东西。
- **它是「历史最深的那个坑」，不是「现在浮亏多少」。** 一段行情里可能有好几个坑，MDD 记录的是**最深的那一个**。
- **它是「已实现的最坏体验」。** 波动率是对未来不确定性的估计；最大回撤是**已经发生过、你真金白银承受过的最痛时刻**。

## 二、真正的杀手：回本非对称性（The Math of Getting Even）

这是本篇**最反直觉、也最该刻进骨头里**的一条。很多人以为「跌 50%、再涨 50% 就回本了」——**大错特错。**

跌下去用的是**峰顶**做分母，爬回来用的是**谷底**做分母，两个分母不一样，于是：

$$ \text{回本所需涨幅} = \frac{1}{1 - \text{回撤幅度}} - 1 $$

跌得越深，回本所需的涨幅**不是线性、而是加速地**变大：

| 回撤幅度 | 回本所需涨幅 | 直觉 |
|---|---|---|
| −10% | +11.1% | 还算对称，问题不大 |
| −20% | +25% | 开始不对称了 |
| −33% | +50% | 跌三分之一，要涨一半 |
| **−50%** | **+100%** | **腰斩，得翻倍才回本** |
| −67% | +200% | 要涨两倍 |
| −80% | +400% | 要涨四倍 |
| −90% | +900% | 几乎……回不来了 |

看懂这张表，你就懂了两件生死攸关的事：

1. **深坑的可怕，是「几何级」的。** −50% 要翻倍，−90% 要涨九倍。这就是为什么 Topic 24 说「**永久性损失**」比「波动」致命——一次 −90% 的重创，可能用**十几年**都爬不回来（日经 1989 年见顶后花了 **30 多年** 才收复，就是活教材）。
2. **控制回撤 = 保护复利。** 复利（Topic 4）最怕的就是深坑——你辛苦滚了十年的雪球，一个 −50% 能把它打回原形，还要再翻倍才勉强追平。**避免大坑，本身就是一种极高的回报。**

> 🔑 **一句话**：涨和跌是不对称的赌局。**同样的百分比，跌的杀伤力远大于涨的修复力。** 所以对长期投资者，「少挖大坑」往往比「多抢涨幅」更重要。

## 三、📉 互动：亲手体验「回本的陡坡」

下面这个工具把上面那张表变成一条你能拖出来的曲线。**拖动「最大回撤」**，看回本所需的涨幅如何**加速飙升**；再设定一个「预期年化回报」，估算**理论上要熬多少年才能爬回峰顶**——感受一下深坑对时间的吞噬。

<script>
(function(){
  var ddEl=document.getElementById('dd'), rEl=document.getElementById('r');
  var svg=document.getElementById('chart'), out=document.getElementById('out'), hint=document.getElementById('hint');
  if(!ddEl||!rEl||!svg||!out) return;
  var W=440, H=210, padL=46, padR=14, padT=16, padB=32;
  var ddMax=90;

  // 回本所需涨幅 = 1/(1-d) - 1
  function recovery(d){ return 1/(1-d) - 1; } // d in [0,1)

  function X(d){ return padL + (d/ddMax)*(W-padL-padR); } // d in %(0..90)
  function Y(g){ // g = 回本涨幅 %, 上限设 400% 以内可视
    var top=400;
    var gg=Math.min(g, top);
    return padT + (1 - gg/top)*(H-padT-padB);
  }

  function draw(){
    var d=+ddEl.value, r=+rEl.value;
    document.getElementById('ddVal').textContent=d;
    document.getElementById('rVal').textContent=r;

    var g = recovery(d/100)*100; // 回本所需涨幅 %
    // 熬多少年： (1+r/100)^t = 1/(1-d/100)  => t = ln(1/(1-d))/ln(1+r)
    var years = Math.log(1/(1-d/100)) / Math.log(1+r/100);

    // 曲线
    var pts=[];
    for(var k=5;k<=ddMax;k++){ pts.push(X(k)+','+Y(recovery(k/100)*100)); }

    var cx=X(d), cy=Y(g);

    // y轴刻度
    var ticks='';
    [0,100,200,300,400].forEach(function(t){
      var yy=Y(t);
      ticks+='<line x1="'+padL+'" y1="'+yy+'" x2="'+(W-padR)+'" y2="'+yy+'" stroke="var(--text-soft)" stroke-opacity="0.15" stroke-width="1"/>'
           + '<text x="'+(padL-6)+'" y="'+(yy+3)+'" font-size="10" fill="var(--text-soft)" text-anchor="end">+'+t+'%</text>';
    });
    // x轴刻度
    [10,30,50,70,90].forEach(function(t){
      var xx=X(t);
      ticks+='<text x="'+xx+'" y="'+(H-8)+'" font-size="10" fill="var(--text-soft)" text-anchor="middle">-'+t+'%</text>';
    });

    svg.innerHTML =
      ticks
      + '<polyline points="'+pts.join(' ')+'" fill="none" stroke="#dc2626" stroke-width="2.5"/>'
      + '<line x1="'+cx+'" y1="'+padT+'" x2="'+cx+'" y2="'+(H-padB)+'" stroke="#2563eb" stroke-opacity="0.35" stroke-width="1" stroke-dasharray="4 3"/>'
      + '<circle cx="'+cx+'" cy="'+cy+'" r="5" fill="#dc2626"/>'
      + '<text x="'+padL+'" y="'+(padT+2)+'" font-size="10" fill="var(--text-soft)">回本所需涨幅 ↑</text>';

    out.innerHTML='账户跌去 <b style="color:#dc2626">'+d+'%</b>，回本需上涨 <b style="color:#dc2626">+'+g.toFixed(0)+'%</b>'
      +'　｜　按 '+r+'% 年化，约需熬 <b>'+years.toFixed(1)+'</b> 年才回到峰顶';

    var msg;
    if(d<=20){
      msg='这个深度还比较「温和」，回本涨幅和跌幅差得不多——正常牛熊里常见，扛住不动通常就能过去。';
    } else if(d<=40){
      msg='坑开始变陡了：不对称性明显起来，回本要花的力气明显大于跌下去。历史上普通熊市大多落在这个区间。';
    } else if(d<=60){
      msg='这就是「腰斩级」重创（2008 全球金融危机中标普 500 约 -57%）。要翻倍才回本，熬的年头以「好几年」计——这一档最容易让人在坑底崩溃割肉。';
    } else {
      msg='深渊级回撤（如个股暴雷、日经泡沫、单一高风险资产崩盘）。回本涨幅几何式飙升、熬的年头以「十年」计——这已不只是波动，而是逼近 Topic 24 说的「永久性损失」。';
    }
    hint.innerHTML=msg;
  }
  ddEl.addEventListener('input',draw);
  rEl.addEventListener('input',draw);
  draw();
})();
</script>

拖完这个工具，两件事应该扎进心里：

1. **曲线不是直线，是一道越来越陡的悬崖。** 前半段（浅坑）还平缓，过了 −50% 之后**几乎垂直向上**——这就是深坑最狰狞的地方。
2. **深坑吞噬的不只是钱，还有「时间」。** 回本要熬的年头会随坑深急剧拉长——而**时间**，正是复利最宝贵的燃料。保住不挖大坑，等于保住了你的复利引擎。

## 四、从数学跨到人性：那个决定成败的「坑底时刻」

到这里，数学部分讲完了。但**真正决定你财富结局的，不是这些公式，而是你在坑底那一刻按下的是哪个按钮。** 这就要请出行为金融学里最著名的一条规律——**损失厌恶（Loss Aversion）**。

卡尼曼与特沃斯基的经典研究发现：**亏 100 块的痛，大约是赚 100 块的爽的 2~2.5 倍。** 这条深植于人类进化的本能，在坑底会演成一整套「自毁剧本」：

- **峰顶时（贪婪）**：账户天天新高，你觉得自己「风险承受力很强」，甚至加杠杆、追高。**此时你严重高估了自己能扛的痛。**
- **下跌初期（否认）**：「就是正常回调，很快就回来。」——于是不减仓、也不检视。
- **深跌中（恐惧 → 恐慌）**：账户腰斩，每天睁眼就是浮亏，媒体一片「崩盘」「归零」。损失厌恶被拉满，**大脑的唯一诉求变成「让这种痛停下来」**——于是在**最不该卖**的坑底附近，割肉离场。
- **卖在坑底之后（永久性损失）**：**「浮亏」被你亲手变成了「实亏」。** 你不但吃满了整段下跌，还**踏空了随后的反弹**——把 Topic 24 里那个可修复的「波动」，亲手办成了不可逆的「永久性损失」。

> 🔑 **这就是那个鸿沟**：牛市里你以为自己能扛 −50%，可真到 −50% 那天，让你做决定的**不是那个冷静的你，而是被损失厌恶劫持的杏仁核**。**风险承受力从来不是你「以为」的数字，而是你「真到坑底」会做出的动作。**

一个冰冷的数据能说明代价有多大：多项研究（如 DALBAR）长期发现，普通基金投资者的**实际收益**，常年**大幅落后**于他们所买基金本身的收益——差距的主要来源，就是**在恐慌中低卖、在狂热中高买**。市场没坑他们，**他们在坑底的动作坑了自己。**

## 五、把「扛得住」变成可操作的功课

既然靠意志力硬扛在坑底几乎必败，成熟投资者的做法是**不依赖坑底的自己**，而是提前用制度把手绑住。四个实操杠杆：

- **① 用「最大回撤」而非「波动率」来体检自己。** 别只问「我能接受多大波动」，要具体地问：「**如果账户从今天的高点跌去 40%、并且趴在坑里两三年，我还睡得着、且绝不卖吗？**」把这个问题**用具体金额**问一遍（比如「$300{,}000 跌成 $180{,}000」），答案会诚实得多。这正是下一篇 Topic 30 要系统做的事。
- **② 用「资产配置」先天削浅坑。** 回撤深度很大程度由持仓结构决定：100% 押股票，历史最深坑可达 −50% 以上；掺入债券与现金后（模块九），坑会明显变浅。**你能扛多深的坑，就该反推你该配多少风险资产**——而不是反过来，先满仓再祈祷自己扛得住。
- **③ 用「机械纪律」替代「坑底决策」。** 定投（定期定额，模块十）、再平衡（模块十二）、预先写下的投资纪律（IPS，模块二十一），本质都是**把决策从「情绪最失控的当下」搬到「情绪最冷静的平时」**——让规则替你在坑底扣动扳机（往往是买入，而非割肉）。
- **④ 先备好「弹药」与「缓冲」，才敢谈扛。** 应急备用金（Topic 11）和不加杠杆，是扛住回撤的物理前提：**手里有余粮、身上没负债的人，才有资格「什么都不做」地扛过去；**而被追加保证金或急等用钱的人，会被迫在最坏的时点卖出——回撤对他就成了永久性损失。

## 六、对投资者意味着什么？

- **回撤是你真正的风险体验，波动率只是它的预告片。** 决定你会不会在半路割肉的，是那个**深不见底的坑**，不是平时的上下颠簸。评估任何策略，都该看它**历史最大回撤**有多深、趴了多久。
- **敬畏非对称性：防守就是最好的进攻。** 跌 50% 要涨 100% 才回本。**少挖一个大坑，胜过多抢几次涨幅**——尤其对靠复利慢慢滚雪球的长期投资者。
- **诚实面对那个鸿沟。** 你「以为」的风险承受力，几乎总是高于「真实」的。**按牛市里的自信去配仓，会在熊市坑底原形毕露。** 宁可保守一档，配一个你在最坏时刻也拿得住的组合。
- **⚠️ 别在坑底把「波动」办成「永久损失」。** 市场的大坑，历史上绝大多数最终会被填平（分散良好的全市场组合尤其如此）；真正不可逆的，是**你在坑底亲手按下的「卖出」**。扛得住，本质是「**不把浮亏变实亏**」。
- **用制度代替意志力。** 提前定好配置、定投、再平衡与纪律，让规则在坑底替你做决定——这比指望「到时候我一定冷静」靠谱一万倍。

## 📝 本篇小结

- **最大回撤（MDD）**：从历史峰顶跌到之后谷底的最大跌幅，衡量「你曾拥有又眼看蒸发掉」的最深那个坑——它是**已实现的最痛体验**，比波动率更贴近真实风险感受。
- **回本非对称性**：回本涨幅 $=\frac{1}{1-d}-1$，跌得越深、回本越难且**几何式变陡**：−50% 要 +100%，−90% 要 +900%。**控制回撤就是保护复利与时间。**
- **损失厌恶**：亏的痛约为赚的爽的 2 倍多；它在坑底把人逼向「割肉止痛」，从而把可修复的**波动**办成不可逆的**永久性损失**。
- **那个鸿沟**：牛市自估的风险承受力 ≫ 熊市坑底的真实行为。**风险承受力是你真到坑底会做的动作，不是你以为的数字。**
- **可操作的功课**：用回撤（而非波动）+ 具体金额体检自己 → 用资产配置先天削浅坑 → 用机械纪律替代坑底决策 → 用应急金和零杠杆备好缓冲。

> **下一篇预告 · Topic 30：🛠️ 评估你的风险承受能力**——这一篇我们发现「你以为能扛的」和「你真会做的」之间有条鸿沟。下一篇就把它落地：手把手区分风险承受力的两个维度——**能力**（财务上你到底亏得起多少：收入稳定性、时间跨度、应急金、负债）与**意愿**（心理上你受得了多深的坑、几年不回本）。两者取其**较低者**，才是你真正该配的风险水平。
